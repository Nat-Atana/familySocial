# Family Social Graph

A responsive React reproduction of the supplied family-network design.

## Run locally

```bash
npm install
npm run dev
```

Create a production build with:

```bash
npm run build
```

## Implementation

- React and Vite
- Semantic HTML cards and labels
- SVG relationship lines and network icon
- Data-driven people and relationships in `src/data.js`
- `react-icons` for familiar interface symbols
- Responsive scaling for the fixed network composition

The dummy avatar URLs can be replaced with local or API-provided profile images without changing the graph layout.
